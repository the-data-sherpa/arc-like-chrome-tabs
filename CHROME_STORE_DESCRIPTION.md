**How to Use:**
1. Click the Tabula icon or press `Ctrl+Shift+E` to open the sidebar
2. Create a workspace using the `+` button
3. Right-click any tab to pin it or add to favorites
4. Organize tabs into folders for better structure
5. Switch between workspaces to manage different projects

**Requirements:**
- Chrome browser (Manifest V3 compatible)
- Permissions: tabs, storage, contextMenus, sidePanel (all required for core functionality)
